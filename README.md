# PortusSophia™ WebKernel

This repository hosts the public WebKernel for PortusSophia™ — the educational, interpretive, and structural scaffolding that surrounds the canonical architecture.

The live site is published at:

https://portussophia.github.io

Content includes:

- WebKernel layout templates
- Interpretive guides
- Witness cycle indices
- Structural recipes
- Public-facing philosophies and scaffolds

This repo does **not** contain canonical governance documents.
Those remain under protected repositories in the PortusSophia™ organization.
